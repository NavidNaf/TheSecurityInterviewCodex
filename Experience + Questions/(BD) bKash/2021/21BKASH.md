## Details

This was an assessment I prepared for the interview process at bKash Fintech Limited. The overall process was divided into four stages:

1. Resume screening
2. Technical assessment
3. Presentation on the submitted solutions
4. HR and final non-technical interview

## Assessment Questions

### Can You VA?

We provided the `https://google-gruyere.appspot.com/` website and asked for a vulnerability assessment. The task was to perform manual penetration testing and submit a detailed report.

### Find Your Way to the Flag

This was a CTF-style challenge focused on website source code auditing. It also required using Python to manually create or work with a library to find the flag.

### Are You OSINTELLIGENT?

This was an OSINT task based on an email address. The goal was to discover any useful publicly available information and report the findings.

### Who's That APK?

We provided a vulnerable APK file, but the goal was only to decompile the APK. No vulnerability identification was required for this part.
