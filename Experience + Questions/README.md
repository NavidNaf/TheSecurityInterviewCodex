# Experience + Questions

This directory stores interview experiences, company-specific security questions, technical assessments, and standalone practice questions.

## Basic Structure

The directory is organized in two main ways:

- Company-based directories
- Miscellaneous question files

Company-based directories usually follow this structure:

```text
Company Name/
└── Year/
    ├── Interview note
    ├── ANSWERS.md
    ├── Source code or assessment files
    └── Solution files, if needed
```

## Company-Based Directories

Company-based folders contain notes from a specific interview process or assessment. These files usually include role details, interview rounds, questions asked, technical tasks, source code, and answers when available.

## Miscellaneous Questions

`miscQuestions.md` is for standalone questions that do not belong to one full company interview write-up. These can be questions collected from different rounds, informal practice, isolated interview prompts, or questions that are useful but do not need their own company folder.

`miscAnswers.md` contains short answers for the questions in `miscQuestions.md`.

## Difference Between Misc and Company Notes

Company-based directories are tied to a specific organization, year, role, or assessment. `miscQuestions.md` is more general and is used for loose questions that are still useful for preparation but do not have enough context for a full interview experience page.
