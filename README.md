# The Security Interview Codex

The Security Interview Codex is a repository for collecting notes from security job interviews. It is meant to be a practical guide for security engineering, security research, and related technical interview preparation.

This repo includes interview experiences, common security questions, technical topics, coding/security review examples, and questions I have prepared or asked in interviews. The goal is to build a comprehensive reference that can help with understanding what security interviews usually look like and how to prepare for them.

## What This Repository Covers

- Security engineering interview experiences
- Security research interview questions
- Technical security concepts and explanations
- Secure coding and code review questions
- Architecture and threat modeling style questions
- CTF-style and vulnerability analysis questions
- Interview questions I have created or used for practice

## Repository Goal

The main goal is to keep all security interview notes in one organized place. Over time, this repository should become a useful guide for anyone preparing for security engineering or security research roles.

## Structure

Interview notes are organized by company, role, or topic when possible. Each section may include the role details, interview format, questions asked, code samples, and short explanations or answers.
